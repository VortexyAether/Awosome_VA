# AI Coding & Lab Adoption

## Shared AI coding agent strategy

- Source: Slack discussion about Claude Max / shared API / student usage.
- Main ideas:
  - Expensive personal Max plans could be shared for hard workloads, but account-sharing and terms should be checked carefully.
  - A shared API budget may be more controllable because extra usage can be topped up and monitored.
  - Students should first learn individual/student-accessible coding agents, then use shared resources for heavier tasks.
- Why it matters:
  - Lab-wide AI coding workflows need both technical onboarding and cost governance.
  - A practical split may be:
    - individual tools for daily coding and learning
    - shared high-capacity account/API for heavy, occasional jobs
