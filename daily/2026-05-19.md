## Added Resources

### Neural operators, SciML foundations & surrogate reliability

- [Martingale Neural Operators: Learning Stochastic Marginals via Doob-Meyer Factorization](https://arxiv.org/abs/2605.15806) — deterministic neural operator가 stochastic PDE에서 conditional mean으로 붕괴하며 variance/tail structure를 잃는 문제를 Doob-Meyer factorization으로 다룬다. UQ가 필요한 thermal/fluid surrogate에서 “평균 field 예측”을 넘어설 근거가 된다. `#neural-operator` `#stochastic-PDE` `#UQ` `#SciML`
- [AOT-POT: Adaptive Operator Transformation for Large-Scale PDE Pre-training](https://arxiv.org/abs/2605.15793) — 다양한 PDE dataset을 한 foundation surrogate로 pretrain할 때 operator structure 차이가 병목이 되는 문제를 adaptive transformation으로 완화하려는 논문. multi-physics pretraining 실험을 설계할 때 데이터 통합 방식을 점검하게 만든다. `#operator-learning` `#PDE-pretraining` `#foundation-model`
- [Eradicating Negative Transfer in Multi-Physics Foundation Models via Sparse Mixture-of-Experts Routing](https://arxiv.org/abs/2605.15179) — open-channel flow, porous media처럼 서로 다른 PDE regime을 dense model에 함께 넣으면 gradient conflict와 plasticity loss가 생긴다는 문제를 sparse MoE routing으로 다룬다. CFD/thermal/porous multi-domain surrogate를 하나로 묶을 때 중요한 경고다. `#multi-physics` `#MoE` `#negative-transfer` `#neural-operator`
- [A numerical study into neural network surrogate model performance for uncertainty propagation](https://arxiv.org/abs/2605.16078) — boundary-value problem solution field surrogate를 stochastic uncertainty propagation 관점에서 평가한다. 설계변수/물성/BC uncertainty가 있는 heat-transfer·CFD surrogate에서 surrogate error가 UQ 결과를 어떻게 왜곡하는지 볼 때 유용하다. `#surrogate` `#uncertainty-propagation` `#validation`
- [Curriculum Learning of Physics-Informed Neural Networks based on Spatial Correlation](https://arxiv.org/abs/2605.15254) — PINN 학습에서 spatial correlation 기반 curriculum을 사용해 high-dimensional nonlinear PDE 학습 난점을 줄이려 한다. fluid mechanics, heat transfer, solid mechanics를 직접 언급해 PINN training protocol reference로 쓸 만하다. `#PINN` `#curriculum-learning` `#heat-transfer` `#fluid-mechanics`

### CAD, MCP & intelligent engineering workflow tools

- [cadgate](https://github.com/vericontext/cadgate) — CadQuery/build123d로 생성된 CAD-as-code PR을 geometric metric diff, minimum wall thickness, DFM rule, 6-view rendered preview로 검사하는 CLI/GitHub Action. CAD agent workflow에서 “생성”보다 중요한 검증 gate 후보. `#CAD` `#validation` `#CadQuery` `#GitHub-Actions`
- [manifold-mcp](https://github.com/zhicwan/manifold-mcp) — manifold-3d 기반으로 LLM-generated TypeScript geometry snippet을 validate하고 live three.js preview 및 STL/3MF export로 연결하는 MCP server. lightweight text-to-manufacturable-geometry 실험에 맞는 agent surface다. `#MCP` `#CAD` `#STL` `#3D-printing`
- [blwfish/freecad-mcp](https://github.com/blwfish/freecad-mcp) — FreeCAD용 32-tool MCP server. 이미 여러 FreeCAD MCP가 있지만, tool coverage가 비교적 명시적이라 parametric part → export → mesh precheck 같은 작은 CAD-to-CAE benchmark에 넣어볼 후보. `#FreeCAD` `#MCP` `#CAD-automation`
- [viznoir](https://github.com/kimimgo/viznoir) — “VTK is all you need”를 표방하는 AI-ready VTK visualization 도구. CFD/CAE agent가 결과를 숫자 로그만이 아니라 VTK artifact와 시각 evidence로 검토하게 만드는 workflow 후보. `#VTK` `#visualization` `#CAE` `#agentic-workflow`

## Possible Research Ideas

1. **Surrogate UQ sanity check:** 간단한 heat equation 또는 lid-driven cavity에서 입력 uncertainty를 넣고 FNO/PINN/MLP surrogate가 mean뿐 아니라 variance, tail, Sobol-like sensitivity를 얼마나 보존하는지 비교한다.
2. **CAD validation contract v0:** CadQuery/build123d/FreeCAD 중 하나로 heat-sink나 duct를 생성하고, cadgate류의 checks + custom meshability/export checks를 PR gate처럼 실행한다.
3. **Multi-physics negative-transfer probe:** thermal diffusion, porous flow, incompressible flow toy datasets를 하나의 operator model에 섞어 학습하고 dense vs sparse expert routing의 regime별 error를 비교한다.
