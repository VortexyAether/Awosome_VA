# Daily Research Radar — 2026-05-09

## Highlights

- **CFD/SciML surrogate는 “빠른 생성”과 “물리 충실도”를 동시에 요구하는 방향으로 가고 있다.** one-step flow reconstruction, one-step generative rollout, context-conditioned flux neural operator가 모두 latency와 long-rollout stability를 정면으로 다룬다.
- **Geometry는 neural operator의 부가 입력이 아니라 지속적으로 보존해야 하는 상태다.** geometric forgetting 논문은 CAD/mesh 기반 CFD surrogate에서 깊은 operator architecture를 검증할 때 좋은 safety check가 된다.
- **3D aerodynamic surrogate는 full-field 직접 예측보다 latent representation + implicit decoder 쪽으로 이동 중이다.** AeroJEPA는 대규모 3D boundary-layer field와 design optimization을 함께 겨냥한다.
- **CAD automation은 discrete command/token, tool-use policy, dynamic assembly로 세분화되고 있다.** Cascaded discrete diffusion과 AADvark는 “보이는 형상”보다 valid command sequence, moving-part interaction, editability를 강조한다.
- **MCP/CAD bridge는 작은 신호지만 계속 늘고 있다.** KOMPAS-3D MCP client 같은 프로젝트는 당장 lab 표준은 아니어도 CAD agent interface 설계 방향을 보여준다.

## Added Resources

### SciML, neural operators & CFD surrogates

- [Physical Fidelity Reconstruction via Improved Consistency-Distilled Flow Matching for Dynamical Systems](https://arxiv.org/abs/2605.05975) — low-fidelity 관측에서 high-fidelity flow field를 one-step consistency model로 재구성. Smoke Buoyancy, Turbulent Channel Flow, Kolmogorov Flow에서 iterative sampling 비용을 줄이는 방향이라 real-time visualization / simulation-in-the-loop에 중요하다.
- [Do Neural Operators Forget Geometry?](https://arxiv.org/abs/2605.05862) — 깊은 neural operator가 layer가 깊어질수록 domain geometry 정보를 잃는다는 “geometric forgetting”을 제시하고 geometry memory injection을 제안. irregular geometry/CAD-to-CFD surrogate 검증에 좋은 경고 신호다.
- [AeroJEPA](https://arxiv.org/abs/2605.05586) — geometry/operating condition context에서 aerodynamic field latent를 예측하고 implicit decoder로 field를 복원하는 JEPA-style 3D aerodynamic surrogate. HiLiftAeroML/SuperWing 같은 큰 field와 design optimization을 겨냥한다.
- [MeLISA: One-Step Generative Modeling for Autoregressive Dynamical System Forecasting](https://arxiv.org/abs/2605.05540) — turbulent regime에서 neural operator drift와 diffusion sampling cost 사이의 trade-off를 one-step MeanFlow autoregressive surrogate로 줄이려는 접근.
- [Context Flux Neural Operator](https://arxiv.org/abs/2605.05488) — finite-volume style Flux NO에 recurrent ViT context injection을 붙여 governing equation/PDE coefficient를 명시적으로 보지 않고 conservation law family를 풀도록 설계. Code: <https://github.com/xx257xx/CONTEXT_FLUX_NO>.

### CAD, design automation & engineering agents

- [Computer-Aided Design Generation by Cascaded Discrete Diffusion Model](https://arxiv.org/abs/2605.05031) — CAD command와 parameter를 discrete diffusion으로 나누어 생성. CAD token의 categorical/discrete nature를 존중한다는 점에서 continuous embedding diffusion보다 engineering-valid sequence에 가까운 방향이다.
- [FeatureFox](https://arxiv.org/abs/2604.26770) — B-Rep CAD machining feature recognition을 panoptic graph segmentation으로 처리하고, 적은 training parts로 높은 PQ를 보임. CAD/CAM automation과 manufacturability-aware CAE preprocessing에 유용한 신호.
- [Agent-Aided Design for Dynamic CAD Models / AADvark](https://arxiv.org/abs/2604.15184) — 움직이는 assembly와 degree-of-freedom을 다루는 agentic CAD system prototype. piston/pendulum/scissors 같은 dynamic assembly가 CAD agent의 다음 병목임을 잘 보여준다.
- [TOOLCAD](https://arxiv.org/abs/2604.07960) — LLM을 CAD engine을 쓰는 tool-using agent로 훈련하고, CAD-CoT와 online curriculum RL을 제안. FreeCAD/CadQuery/OpenSCAD agent workflow의 paper anchor로 좋다.
- [KOMPAS-3D MCP client](https://github.com/dwnmf/KOMPAS-3D-MCP-bin) — KOMPAS-3D를 MCP에 연결하는 Windows client. 특정 CAD 생태계용이지만 agent-readable CAD control surface가 늘고 있다는 workflow 신호다.

## Best Find of the Day

**Do Neural Operators Forget Geometry?**가 오늘의 best pick. CFD/CAE surrogate에서 geometry는 한 번 encode하면 끝나는 metadata가 아니라, rollout과 depth 전반에 걸쳐 계속 유지되어야 하는 constraint다. VA가 CAD-to-CFD surrogate나 mesh-aware neural operator를 볼 때 “geometry retention probe”를 validation checklist에 넣을 만하다.

## Possible Research Ideas

1. **Geometry retention benchmark:** CAD/mesh 기반 CFD surrogate에서 layer-wise probing으로 geometry 정보가 얼마나 유지되는지 측정하고, prediction error/stability와 상관관계를 본다.
2. **One-step reconstruction for CFD post-processing:** coarse OpenFOAM 결과 → high-resolution field reconstruction을 flow matching / consistency distillation 계열로 실험해 visualization latency를 줄인다.
3. **Agentic CAD validation loop:** AADvark/TOOLCAD/CADFS류를 FreeCAD or CadQuery MCP와 연결해 “generated CAD → DOF/constraint check → meshability check → simple CFD/thermal case”까지 자동 검증한다.
