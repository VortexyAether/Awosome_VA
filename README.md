# Awosome_VA

A curated knowledge base for **CFD × Scientific Machine Learning × AI-assisted engineering**.

This repository is maintained as a lightweight, categorized index of useful papers, tools, courses, repositories, and technical notes. The focus is on resources that may help with CFD-ML research, differentiable simulation, neural operators, meshing, local AI workflows, and safe research infrastructure.

## Curation rules

- Prefer resources directly useful for CFD, Scientific ML, CAE, differentiable simulation, or research automation.
- Skip casual discussion, assignments, and ephemeral Slack chatter unless it contains a concrete reusable resource.
- Keep each entry short: link, why it matters, and possible use.
- Add new resources under the most specific category first.
